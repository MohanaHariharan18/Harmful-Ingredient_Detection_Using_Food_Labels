# Harmful Ingredient Detection Using Food Labels

Welcome to the **Harmful Ingredient Detection Using Food Labels** repository! 🚀

This project leverages **Artificial Intelligence, Machine Learning, Computer Vision, and Optical Character Recognition (OCR)** to identify potentially harmful ingredients from packaged food labels. The application helps users make informed dietary choices by automatically extracting ingredient information and analyzing it against a database of allergens, preservatives, and harmful additives.

---

## 🏗️ Project Architecture

The project follows a modular AI/ML pipeline:

* **Image Acquisition:** Upload or capture food label images.
* **OCR Processing:** Extract ingredient text using OCR.
* **Data Preprocessing:** Clean and standardize extracted text.
* **Ingredient Analysis:** Compare ingredients against a harmful ingredient database using machine learning and rule-based classification.
* **Result Generation:** Display detected harmful ingredients along with health risk information.
